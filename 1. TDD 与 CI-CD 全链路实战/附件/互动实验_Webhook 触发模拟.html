1. 事件驱动 (EDA)

由推送行为主动触发，而非服务器轮询 Git 仓库状态。

2. 异步解耦

Git 仓库发出 Webhook 后即刻结束，不阻塞后续本地开发任务。

3. 幂等性与状态

CI 接收端立即返回 \( 202 \) 确认接收，随后再执行长耗时的构建任务。
