## 二、必须完成的操作任务（70分）

### 任务1：Git环境配置（10分）

**要求完成：**

1. ✅ Git环境配置
   - 完成Git安装、基础配置和SSH密钥配置
   - 提交 `ssh -T git@gitee.com` 的成功输出截图

**评分标准：**
- 完成配置并提交截图：10分
- 未完成：0分

### 任务2：Gitee仓库创建与基本操作（15分）

**要求完成：**

1. ✅ 创建Gitee仓库
   - 仓库名称：`git-practice-学号`（例如：git-practice-2024001）
   - 仓库类型：公开
   - 提交仓库主页截图

2. ✅ 克隆仓库到本地
   - 提交本地仓库目录结构截图

3. ✅ 第一次提交
   - 修改README.md文件，添加个人信息
   - 提交 `git push` 成功截图
   - 提供Gitee仓库的访问链接

**评分标准：**
- 完成所有3项，仓库可访问：15分
- 完成2项，仓库可访问：10分
- 完成1项：5分
- 未完成：0分
